# Petit of the Dead

[![Petit of the Dead](http://img.youtube.com/vi/xQvHxMYSHa8/0.jpg)](https://www.youtube.com/watch?v=xQvHxMYSHa8)

## SIDELOAD
プロジェクトを Arduiono IDE でビルドし、Arduboy に転送してください。

## 操作
- `↑`,`↓`,`←`,`→`<br>前後左右に移動
- `A`+`↓`<br>武器の変更
- `B`<br>発射

## 開発
- [Arduino](https://www.arduino.cc)
