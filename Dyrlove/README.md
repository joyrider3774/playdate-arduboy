# Do You Remember Love?

[![Do You Remember Love?](http://img.youtube.com/vi/W0yt_K_wM2Q/0.jpg)](https://www.youtube.com/watch?v=W0yt_K_wM2Q)

## SIDELOAD
プロジェクトを Arduiono IDE でビルドし、Arduboy に転送してください。

## 操作
- `↑`,`↓`,`←`,`→`<br>上下左右に移動
- `A`<br>左方向に発射
- `B`<br>右方向に発射

## 開発
- [Arduino](https://www.arduino.cc)
